# 🤖 Telegram Shop Bot - Бот для продажи игрушек в ЖК Дубровка

![Telegram Shop Bot](https://img.shields.io/badge/Telegram-Bot-blue?logo=telegram)
![Python](https://img.shields.io/badge/Python-3.8+-green?logo=python)
![License](https://img.shields.io/badge/License-MIT-yellow)

**Telegram Shop Bot** — это полноценный бот для интернет-магазина с корзиной покупок, админ-панелью и красивым сайтом-витриной. Создан специально для продажи игрушек в ЖК Дубровка.

## ✨ Возможности

### 🤖 Бот (bot.py)
- **Каталог товаров** - плюшевые и обычные игрушки с фото и описанием
- **Корзина покупок** - добавляйте, удаляйте, очищайте
- **Оформление заказов** - самовывоз в ЖК Дубровка
- **Система сохранения** - все данные сохраняются автоматически
- **Уведомления админам** - о новых пользователях и заказах

### 👑 Админ-панель
- 📊 **Статистика** - просмотр активности
- 👥 **Управление пользователями**
- 🛒 **Просмотр заказов**
- 📦 **Управление товарами**
- 📢 **Рассылка сообщений**
- 💾 **Ручное сохранение данных**

### 🌐 Сайт-витрина (index.html)
- Автоматическая загрузка товаров из JSON
- Горизонтальная карусель с товарами
- Поиск по названиям
- Фильтрация по категориям
- Адаптивный дизайн
- Красивое отображение с иконками-заглушками
